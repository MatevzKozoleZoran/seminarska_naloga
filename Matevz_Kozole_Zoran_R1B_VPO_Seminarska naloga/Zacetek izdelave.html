Začetek izdelave

Najprej moramo zapisati 2 vrstici kode, ki sta:

  • import pygame
  • pygame.init()

S tem smo ukazali da naj program uporabi knjižnico Pygame.

Nato zapišemo katere barve bi radi uporabljali.Zapisati moramo 3 številke, ki povejo katera barva se prikaže. Zgled za rdečo barvo:

RED = (255, 0, 0)

Jaz sem uporabil 6 barv, ki so: rdeča, oranžna, rumena, svetlo modra, temno modra in bela. Če jih hočemo spremeniti ali dodati, moramo samo zapisati tiste 3 številke.

Dodal sem še 2 spremenljivki: lives in score, ki sem jih uporabil kasneje.

Napisati moramo tudi kodo, da se bo program sploh odprl in se ne bo zaprl, kako veliko bo to okno, ter naslov programa:

  • size = (800, 600)
  • screen = pygame.display.set_mode(size)
  • pygame.display.set_caption("Breakout Game")

Napisati moramo tudi zanko, ki vsebuje 3 glavne oddelke:

  • Kontrole, torej kako se igra upravlja,
  • Kaj se zgodi ko se igro igra,
  • Osvežitev zaslona, da se stvari v igri premikajo

Preden napišemo zanko moramo najprej napisati:

  • carryOn = True, ki nam pove, da se bo zanka odvijala dokler ne zapremo programa.
  • clock = pygame.time.Clock(), ta ukaz se bo uporabljal za kako hitro se bo zaslon posodabljal.

Zanka

Glavni del zanke:

  • While carryOn:
  • for event in pygame.event.get():, se uporablja ko nekaj naredimo
  • if event.type == pygame.QUIT: carryOn = False, če zapremo program se zanka preneha.

Zapišemo barvo ozadja:
screen.fill(DARKBLUE)
pygame.draw.line(screen, WHITE, [0, 38], [800, 38], 2)

Da nam prikaže življenja in točke na vrhu okna zapišemo:
font = pygame.font.Font(None, 34)
text = font.render("Score: " + str(score), 1, WHITE)
screen.blit(text, (20,10))
text = font.render("Lives: " + str(lives), 1, WHITE)
screen.blit(text, (650,10))

Da nam posodobi točke in življenja zapišemo:
pygame.display.flip()

Omejimo FPS na 60:
clock.tick(60)

Ko končamo z zanko, se program zapre:
pygame.quit()
